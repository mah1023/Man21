Главная Метод мат.индукции Похідна Первісна

Що таке індукція?

Означеня: Індукція означає логічний перехід від часткових фактів до деякого загального твердження.

Розглянемо наступний елементарний приклад.

Задача 1.1 Довести, що множина {1,4,9,16,25} складається з чисел які є точнимим квадратами.

Розв'язання. Оскільки 1=12, 4=22, 9=32, 16=42, 25=52, то справді всі числа даної множини є точними квадратами.

У чому ж полягає такий метод доведення? Встановивши, що властивість "бути точним квадратом" виконується для кожного елемента i заданої множини, ми зробили відповідний висновок і про всю множину в цілому. Такий метод доведення називається методом повної індукції. Зрозуміло, що він застосовний, якщо повний перебір можна здійснити за скінченне число операцій, а кількість таких операцій не надто велика.

А як бути, коли досліджувана множина містить нескінченну кількість елементів? Адже тоді повний перебір стає неможливим.

Задача 1.2 Послідовність Xnвизначена формулою Xn=(n-1)*(n-2)*...*(n-2002)+n. Чи вірно, що Xn=n при всіх натуральних n?

Розв'язання. Ні, невірно, бо для n > 2002 матимемо Xn>n

Задача 1.3 Довести, що число n3 - 4n ділиться на 3 при кожному натуральному n.

Розв'язання. Натуральнi числа n при діленні на 3 можуть давати лише остачі 0, 1, 2. При цьому остачу 0 дають числа n=3k, остачу 1 - числа 3k +1, остачу 2 - числа 3k +2, де k - ціле невід'ємне число. Оскільки при кожному такому k числа (3k)3 -4*3k,

(3k+1)3-4*(3k+1)=3*(9k3+9k2-k-1),

(3k +2)3-4*(3k +2)=3*(9k3+18k2+12k) діляться на 3,то n3-4n ділиться на 3 при кожному натуральному n.

Зауважимо, що спроба послідовно надавати числу и значень 1, 2, 3 ... знову не привела б до успіху. Хоча ми при цьому кожного разу діставали б числа, що дiляться на 3, та гарантувати, що така властивість збережеться і надалі, все ж не змогли б.

Що таке метод математичної індукції?

Означення: Метод математичної індукції-це шлях доведення тверджень, що залежать від натурального параметра. Застосування метода математичної індукції грунтується в принципі математичної індукції.

Означення: Твердження А(n), залежне від натурального параметра n, вважається доведеним для всіх натуральних n, якщо воно доведене для n=1 і з припущення про виконання твердження A(k) для якого-небудь натурального k виведено, що справджується твердження А(k+1) для натурального числа k+1.

Доведення: Нехай при деякому n>2 тведження А(n) було неправильним, то ми могли б вибрати із всіх таких n найменше. Нехай таким є n=nо. Оскільки тоді для всіх nо твердження А(n) вірне, то А(n-1) теж правильне.Але в такому разі воно повинно справджуватися і для n=(nо-1)+1=nо. Одержане протиріччя і доводить правильність А(n) для вісх натуральних значень n.

Відзначимо, що доведення правильності твердження А(1) називається базою індукції, а доведення правильності А(k+1) після припущення правильності А(k)- індукційним переходом. В деяких задачах властивість А(n) попередньо може бути не заданою. Припущення про твердження А(n) в такому випадку називаеться індукційним припущенням.

Задача 1.3 методом математичної індукції:

При n=1 маємо:n3-4n=13-4=-3. Це число ділиться на 3. Припустимо, що при n=k число k3-4k теж ділиться на 3. Тоді для n=k+1 маємо:(k+1)3-4*(k+1)=k3+3*k2+ 3*k+1-4*k-4=k3-4*k+3*(k2+k+1).Оскільки за припущенням k3-4*k ділиться на 3, то звідси випливає, то й (k+1)3-4*(k+1) ділиться на 3. Отже на основі принципу математичної індукції число n3-4*n ділиться на 3 при кожному натуральному n.

Задача 1.4: Довести, що при кожному натуральному n cправджуеться рівеість 1+3+5+...+(2n-1)=n².

Розв'язання: При n=1 маємо 1=12. Припустимо, що при n=k правильна рівність 1+3+...+(2k-1)=k2.Тоді при n=k+1 отримаємо 1+3+...+(2k-1)+2*(k+1)-1=k2+2*(k+1)-1=(k+1)2.Отже, індукційний перехід вірний, а тому для всіх натуральних n правильна рівність 1+3+5+...+(2n-1)=n2.
